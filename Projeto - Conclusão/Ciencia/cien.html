Ciência de Dados

Amostragem: Amostragem é um processo utilizado em diversas áreas, como estatística, pesquisa de mercado, ciências sociais e muitas outras, para obter informações sobre uma população maior por meio da seleção de uma parte representativa dessa população. Em vez de coletar dados de todos os indivíduos ou elementos da população, esse permite extrair informações úteis a partir de uma amostra menor.

Amostragem aleatória: Na ciência de dados, a metodologia aleatória é um método utilizado para selecionar uma amostra de dados de um conjunto maior, a fim de realizar análises estatísticas e inferências sobre a população.

Amostragem sistemática: Na ciência de dados, a sistemática é um método de seleção de seleção que envolve a seleção de elementos em intervalos regulares de um conjunto de dados ordenados. É uma técnica utilizada quando a ordem dos elementos no conjunto de dados é importante.

Amostragem estratificada: um método de seleção de amostras em que a população é dividida em subgrupos distintos chamados estratos. Uma amostra é selecionada aleatoriamente dentro de cada estrato, garantindo que cada subgrupo esteja adequadamente representado na amostra final.

Tendência central: conceito estatístico que busca resumir ou descrever o valor central ou típico de um conjunto de dados.

Média: Na ciência de dados, a média é uma medida estatística comumente utilizada para representar o valor central de um conjunto de dados. Também conhecida como média aritmética, ela é mostrada somando todos os valores do conjunto e dividindo pelo número total de observações.

Mediana: Na ciência de dados, a mediana é uma medida estatística que representa o valor central de um conjunto de dados ordenado. É o valor que divide o conjunto de dados em duas partes iguais, ou seja, 50% dos valores estão acima e 50% estão abaixo da mediana.

Moda: Na ciência de dados, a moda é uma medida estatística que representa o valor mais frequente ou com maior ocorrência em um conjunto de dados. Em outras palavras, é o valor que aparece com maior frequência em um conjunto de observações.

Percentil: Na ciência de dados, o percentil é uma medida estatística que indica a posição relativa de um valor em relação a um conjunto de dados. Ele ajuda a entender como um valor específico se compara aos demais valores do conjunto.

Quartil: Na ciência de dados, os quartis são medidas estatísticas que dividem um conjunto de dados ordenado em quatro partes iguais. Eles são úteis para entender a distribuição dos dados e identificar os valores que dividem o conjunto em quartos.

Medida de dispersão: Na ciência de dados, as medidas de dispersão são usadas para quantificar a variabilidade ou dispersão dos valores em um conjunto de dados. Essas medidas fornecem informações sobre a dispersão dos dados ao redor de uma medida central, como a média.

    Amplitude: A amplitude é uma medida simples que representa a diferença entre o maior e o menor valor em um conjunto de dados. No entanto, uma amplitude pode ser influenciada por valores extremos e não fornecer uma medida abrangente da variabilidade dos dados.

    Variância: Na ciência de dados, a variância é uma medida de dispersão que quantifica o quão distantes os valores de um conjunto de dados estão na média. Ela mede a média dos quadrados das diferenças entre cada valor e a média.

    Desvio padrão: Na ciência de dados, o desvio padrão é uma medida de dispersão que quantifica o grau de variação ou dispersão dos valores em um conjunto de dados em relação à média. Ele é calculado como a raiz quadrada da variância.

    Qui quadrado:Na ciência de dados, o teste do qui-quadrado é um teste estatístico utilizado para determinar se existe uma associação significativa entre duas variáveis ​​categóricas em um conjunto de dados.
